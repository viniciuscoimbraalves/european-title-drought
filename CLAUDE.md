# European Title Drought ‚Äî CLAUDE.md

## Project
Website tracking how long it's been since 23 European clubs last won the UEFA Champions League. Real-time counter with years, months, days, hours, minutes, and seconds.

**URL**: https://european-title-drought.vercel.app
**Stack**: React 18 + TypeScript + Vite + TanStack Router v1 + Tailwind CSS v4 + Vercel

## Commands
- `npm run dev` ‚Äî local dev server
- `npm run build` ‚Äî production build
- `npm run preview` ‚Äî preview production build
- `npm run format` ‚Äî Prettier
- `npm run lint` ‚Äî ESLint
- `npx vercel --prod` ‚Äî manual production deploy

## Business rules
- Dates with year ‚â§ 1900 are placeholders (club never won that competition) ‚Äî filtered in components.
- `calculateDrought()` receives ISO date + optional Date ref, returns `Drought` with years/months/days/hours/minutes/seconds/totalDays/totalSeconds.
- Timer updates every 1 second via `setInterval` + `useState<Date | null>` for correct hydration.
- `localStorage.getItem("european-title-drought-theme")` controls light/dark theme with `.dark` class on `<html>`.

## Design System
- **Colors**: oklch (`src/styles.css` lines 64-141), primary ~blue (260 hue), secondary ~purple (300 hue)
- **Gradients**: `--gradient-hero` (blue-purple) used in logo and page headers
- **Animations**: Tailwind transitions, `tw-animate-css`
- **Icons**: `lucide-react`

## Logo
- Inline SVG in `src/components/Logo.tsx` with `currentColor` and `--gradient-hero`
- Trophy/star icon + "European Title Drought" text
- No external images, no HTTP requests, no theme prop

## Code patterns
- Routes: TanStack Router v1 with `createFileRoute` + `Route.useParams()`
- Style: Tailwind CSS v4 with `@theme inline`, `@custom-variant dark`
- Fonts: system-ui, tabular-nums via `.font-timer` class
- Components exported as `export function Name()`
- CSS variables with `oklch()`, no hex or rgb
- Foundation and title dates in ISO format (YYYY-MM-DD)

## Deployment Flow
1. **Local** ‚Äî edit, test (`npm run dev`, `npm run build`)
2. **Commit + Push** to GitHub ‚Äî `git add` + `git commit` + `git push`
3. **Deploy** ‚Äî `npx vercel --prod`

Never deploy without committing and pushing first. The three environments must be kept identical at all times.
